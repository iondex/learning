以 Stanford **CS224N**（NLP with Deep Learning，前置）和 **CS336**（Language Modeling from Scratch）
两门课为主线，沉淀 LLM 各知识域的笔记：推理优化、RL/SFT/对齐、Agents、Systems、Scaling Laws 等。

## 结构

```
00-Inbox/      速记、剪藏、待消化
10-Topics/     ★主题知识库（原子化永久笔记，按知识域）
20-Courses/    课程跟讲笔记（CS224N / CS336）
30-Papers/     论文 / 文章精读
40-Daily/      每日笔记（进度追踪）
90-Templates/  Obsidian 模板
99-Assets/     图片附件
```

- 知识沉淀在 `10-Topics/`，课程笔记把知识点**链接**回主题，避免困在课程里。
- 每日笔记串联当天涉及的所有笔记，形成可回溯的学习轨迹。
