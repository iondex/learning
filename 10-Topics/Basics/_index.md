# Basics 基础知识

学习 LLM 所需的底层基础：深度学习基本概念与数学工具。后续主题都建立在这之上。

## 范围
- **数学**：线性代数（矩阵运算、特征分解）、微积分与矩阵求导、概率统计。
- **深度学习基础**：神经网络、反向传播（backpropagation）、损失函数、优化器（SGD/Adam）、
  正则化（dropout 等）、初始化、激活函数。
- **PyTorch 基础**：张量、autograd、训练循环。
